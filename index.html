What is HTML

HTML is a markup language that is used to introduce structure to a website. HTML uses tags to create elements to tell the browser how to display the content.

Origin

The first version of HTML was written by Tim Berners-Lee in 1993.

Since then, there have been many different versions of HTML. Currently, HTML5 is the leading version used by developers.

HTML5

HTML5 was first released in a public-facing form on 22 January 2008.

Its goals were to:

  • improve the language with support for the latest multimedia and other new features
  • to keep the language easily readable by humans
  • be understood by computers and devices such as web browsers, parsers, etc.
  • avoid XHTML's rigidity
  • to remain backward-compatible with older software.
